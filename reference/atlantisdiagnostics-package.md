# atlantisdiagnostics: Tools for calibrating Atlantis ecosystem models

Tools to aid in diagnosing issues in a parameterized Atlantis Ecosystem
model

## Details

- Determine functional group persistence over time

- Determine functional group stability over time

- Determine functional group biomass (relative to historic data) lie
  within reasonable bounds

- Determine distribution of biomass over age classes

- Determine maximum size (length and weight) of species relative to
  observations
